### Step 1 ###
# Create/copy your desired custom buttons. 
# Keep in mind that they will get displayed as 18px squared, with a 2px black border.
# This can be adjusted below if you know a little html.
### Step 2 ###
# Host your button files on Cloud Files, and input the URLs below as the: 
# src="http://your_cloud_files_url/button_description.jpg" entry for each button
# You'll also need to update the href="URL" so each one so it actually does something when clicked
# the prefixes (i.e. tel: mailto: https: http:) provide the local machine with URI-handling instructions
# Your cell phone provider should have an email address associated with your phone number;
# a little bit of google-fu and testing will help you figure out how to receive texts via email. 
### Step 3 ###
# Once you have the config file updated, test it by making sure it has a .html extension; then
# open it up with your browser. Firefox takes the URI-style: file:///path/to/your/file.html
# Make sure all the buttons look and function as desired before proceeding.
### Step 4 ###
# Use the Racker email signature generator page for the first part of your email signature.
# This step is optional if you have already done this previously and just want to add buttons.
# The current URL is: https://rackspacemarketing.com/signatyourEmail/
# If that does not work, look it up on the one wiki, or ask a co-worker for the current one.
# Email me and I can update this document.
### Step 5 ###
# Go to Outlook > Preferences > Signatures
# copy/paste the buttons from your HTML page (generated by this config file) beneath your signature
### Step 6 ###
# Test your work by sending yourself an email and then checking the functionality/appearance of all buttons.
# Share with your friends!
